Considerations About the Program

I made this dedicated web browser named Playcipe. I made a personnal recipe to add to favorites in Edge for surfing on Youtube with simple changes in HTML. I also made the Edge extension named recipe-extension. You can edit it with some editors I made, named CodeEditor and TextEditor. This file is explaining the way to get playlist id and channel id. This video can help you to create your Youtube data API key, in order to get channel id. The recipe page isn't to use as if, because it's very personal. It's the index.html file under assets folder where the program is. To note, there's handle to keys page up (to scroll up), page down (to scroll down), and F11 (to set Playcipe in fullscreen). It comes with another program, when having a laptop PC and Bluetooth speakers, to enable both. There's a parameters file under same folder of the program if there is a error message, it need to set the parameter to True or False, it's made if first device is the capture device. To note, it ask to set a audio quality of 16 bits and 44100 Hz format for both audio cards, and enable both material acceleration and enhanced audio, under audio settings, but there's also another parameter to disable the program to not automatically launching. It also need to set level of earing for laptop speakers to around 10 for avoid too loud sound. My source codes and program are here.
